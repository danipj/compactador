Estruturas de Dados

O nosso trabalho final está categorizado entre os "manipuladores de arquivos", portanto inicialmente vamos fazer a leitura das informações que temos armazenadas em um arquivo.

Para fazer a leitura dos dados, utilizaremos as classes abaixo:

  • FileInputStream, que nos permite o acesso a um arquivo físico
  • BufferedInputStream, que proverá o stream de dados das informações vindas do arquivo e
  • DataInputStream, que permitirá a manipulação dessas informações.

No início do código então teremos, para abrir um arquivo de nome teste.txt, as seguintes linhas de comando:

        FileInputStream arquivoFisico = new FileInputStream("teste.txt");
        BufferedInputStream buffReader = new BufferedInputStream(arquivoFisico);
        DataInputStream   data = new DataInputStream(buffReader);

Logo após a abertura, declaramos um vetor de byte para armazenar os caracteres que estão no arquivo, uma vez que sabemos que cada caracter ocupa exatamente um byte na memória.

     byte vetByte[];

Para instanciar esse vetor, usaremos um método de FileInputStream chamado available() que nos diz exatamente quantos bytes o arquivo tem. Sendo assim, a instancia do vetor se faz da seguinte forma:

   vetByte = new byte[arquivoFisico.available()];

Fazendo uso do DataInputStream, vamos ler os dados do arquivo para o vetor de bytes da seguinte forma:

   data.read(vetByte);

Para escrever esse vetor de Bytes na tela, vamos usar um for um pouco diferente do que temos usado habiltualmente.

Nesse for, vamos escrever letra por letra, de forma que os caracteres especiais (como pulo de linha) serão impressos como caracteres e nos darão o efeito desejado. Para pegar cada letra, vamos usar um método do tipo String que se chama toCharArray, que converte a String para um vetor de chars.

Tendo o vetor de char, vamos usar a nova notação do for que tem dentro dos parênteses apenas dois elementos separados por dois pontos (:). O elemento antes dos ":" é o elemento que vai receber um valor e o elemento após o ":" é o elemento que vai doar valores, um ou mais, porém um de cada vez. O elemento doador deve ser um conjuto (no nosso caso um vetor de char) e assim o for vai automaticamente percorrer o vetor e colocar a cada iteração, um valor no elemento antes dos ":". Assim não precisamos mais inicializar um variável e controlá-la para percorrer o vetor!!

Mas atenção!! Para usar o for desta forma, precisamos garantir que todas as posições dele estejam ocupadas ou tenham elementosa a serem trabalhados.

Veja como fica o código:

        for (char c : new String(vetByte).toCharArray())
        {
            System.out.print(c);
        }

Coloque tudo isso dentro de um main e rode pra ver o resultado!!!

Essa é apenas uma forma de trabalhar com arquivo, mas ela lhe será muito útil!!!

Agora que já temos a especificação do trabalho com arquivo, faça um método que realize a leitura de um arquivo e crie uma tabela de aparições dos caracteres nele contidos. O resultado final é a quantidade de vezes que cada caracter aparece no arquivo.

Para isso, use a estrutura que vc desejar, desde que ao final, possamos saber quais são os caracteres que aparecem no arquivos e quantas vezes ele aparece.

Depois de conseguir esses dados, ordene-os de forma que os caracteres que aparecem mais vezes sejam os primeiros da estrutura!
